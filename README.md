# Proyecto libre iOS avanzado – Recetas saludables

Autor: Carlos Alfredo Callagua Llaque**

App universal de recetas saludables para iPhone y iPad.  
Muestra una lista de recetas y una pantalla de detalle con:

- Nombre
- Categoría
- Dificultad
- Tiempo
- Calorías
- Imagen
- Ingredientes
- Pasos

Implementado con UIKit, UITableView, Navigation Controller y vistas de detalle adaptadas mediante Auto Layout.
